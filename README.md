# Research Report Reader Skill（投研报告精读助手）

一个用于系统化阅读和分析投资研究报告的 Claude Skill。

## 功能

- **三层阅读法提炼**：逐篇提取核心观点、关键假设、投资建议和风险提示
- **跨报告交叉比对**：识别一致性信号、矛盾点和"预期差地图"
- **精读推荐与方法指导**：推荐最值得精读的报告，并给出针对性阅读方法（宏观框架类/微观调研类/行业公司类）
- **英文报告全文翻译**：自动检测英文报告并生成逐字逐句的中文翻译，还原表格和格式

## 使用方法

1. 将 research-report-reader.skill 安装到你的 Claude Skill 库中
2. 上传 PDF 研究报告
3. 用自然语言触发，如"帮我读一下这些研报"、"分析这几篇报告"、"研报精读"等

## 文件说明

- SKILL.md — Skill 的完整指令文件
- research-report-reader.skill — 打包好的 Skill 安装文件（可直接安装）
